BACKEND: (es como la parte de atras de los procesos de una pagina o programa)
Rendimiento
Logica del servidor
Base de Datos
Microservicios
Programa API
Seguridad

FRONTEND: (es lo que esta en la parte delantera de una pagina o programa)
Interfaz web
Logica del cliente
Consumo de APIS
Interaccion de usuario

APIS: (se encarga de unir el BACKEND con el FRONTEND)

REST > API > GRAPHQL

EJEMPLO EN UN RESTAURANTE:
BACKEND es la cocina, FRONTEND es la mesa y otras cosas visuales del RESTAURANTE, API serian los meseros

HTML: da la estructura a la pagina
JAVASCRIPT: hace mas dinamica la pagina

Antes de hacer la pagina siempre se debe hacer la enmaquetacion de esta

Primera practica en HTML:
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Ejemplo basico HTML</title>
</head>
<body>
    <h1>Pagina HTML</h1>
</body>
</html>

Bootstrap: https://getbootstrap.com/
Bootstrap:Es una framework para dar estilo a la estructura de HTML
Con esta linea me enlaza con bootstrap de forma que traer todas sus librerias:
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0-alpha1/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-GLhlTQ8iRABdZLl6O3oVMWSktQOp6b7In1Zl3/Jr59b6EGGoI1aFkw7cmDA6j6gD" crossorigin="anonymous">
Poner el link anterior debajo de la etiqueta 

Por ejemplo para una etiqueta button se busca en botones de la pagina de bootstrap: https://getbootstrap.com/docs/5.3/components/buttons/
Luego se encuentra: class="btn btn-outline-primary"
y otros estilos se copia esa parte al boton requerido de forma que queda:
<button type="button" class="btn btn-outline-primary">Primary</button>
